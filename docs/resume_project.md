# 简历项目描述

## 项目名称

B站爆款视频弹幕互动与评论情绪分析

## 简历写法

- 使用 Python 采集 B 站影视飓风单条爆款公开视频的基础指标、公开弹幕样本和公开评论样本，获取播放量、点赞、投币、收藏、分享、评论、弹幕等互动数据。
- 使用 pandas 对弹幕和评论文本进行清洗、去重、空值处理和字段标准化，将弹幕按 30 秒时间桶聚合，定位高讨论度视频片段。
- 使用 jieba 和 SnowNLP 完成中文分词、关键词提取和情绪倾向分析，构建弹幕关键词、评论关键词、情绪占比、评论情绪与点赞数关系等指标。
- 设计 MySQL 数据表并实现自动建库建表与数据导入，沉淀视频信息、弹幕明细、评论明细、互动指标、时间轴指标、关键词指标和情绪指标。
- 基于 Flask + Echarts 搭建本地交互式可视化看板，展示视频概览、互动率结构、弹幕时间轴、弹幕高峰片段、关键词排名和评论情绪分布。

## 简历精简版

B站爆款视频弹幕互动与评论情绪分析：基于 Python 采集影视飓风公开视频基础指标、弹幕和评论样本，使用 pandas 完成文本清洗与时间桶聚合，结合 jieba / SnowNLP 构建关键词和情绪指标，设计 MySQL 表存储明细及聚合数据，并用 Flask + Echarts 搭建可视化看板，分析视频互动率、弹幕高峰片段和用户评论反馈特征。

## 面试关键词

Python 爬虫、公开接口采集、JSON/XML 解析、pandas 数据清洗、jieba 中文分词、SnowNLP 情绪分析、MySQL 数据建模、Flask API、Echarts 可视化、数据隐私处理。
